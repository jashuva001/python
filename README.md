# python

Collection of basic Python programs by jashuva001.

## About

This repository contains beginner-friendly Python programs demonstrating basic concepts and examples. Each program is a standalone script intended for learning and experimentation.

## Programs

- hello_world.py — Prints "Hello, world!"
- calculator.py — Simple CLI calculator supporting +, -, *, /
- factorial.py — Compute factorial of a non-negative integer
- fibonacci.py — Print first n Fibonacci numbers
- prime_check.py — Check if a number is prime
- palindrome.py — Check whether a string is a palindrome (ignores non-alphanumeric characters and case)
- guess_number.py — A number guessing game (1-100)
- file_io_example.py — Example of writing to and reading from a text file

## How to run

1. Clone the repository:

   git clone https://github.com/jashuva001/python.git

2. Change into the repository directory:

   cd python

3. Run a program with Python 3, for example:

   python3 hello_world.py

Replace `hello_world.py` with the program filename you want to run.
